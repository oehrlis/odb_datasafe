# OraDBA Data Safe Extension v0.10.1

**Release Date:** 2026-02-16

## Overview

Version 0.10.1 is a documentation maintenance release. It reduces recurring
version/date updates across docs and refreshes the main index highlights.

## Highlights

- **Lower maintenance overhead** by avoiding duplicated version/date values in
  top-level docs.
- **Updated key features summary** in `doc/index.md` to match current scope.
- **Version-agnostic tarball examples** in standalone usage documentation.
- **Base64 password input** auto-detected for `ds_database_prereqs.sh`.

## Details

### Changed

- Replaced hardcoded version/date text with stable references to `VERSION` and
  `doc/release_notes/` in:
  - `README.md`
  - `doc/index.md`
  - `doc/quickref.md`
- Updated key features in `doc/index.md` to reflect the latest 0.10.x changes.
- Standardized "Available Scripts" table formatting in `doc/index.md`.
- Updated standalone tarball example in `doc/standalone_usage.md` to use
  `odb_datasafe-<version>.tar.gz`.
- `ds_database_prereqs.sh` now auto-detects base64 input for `--ds-password`.
