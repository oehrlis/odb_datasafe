# OraDBA Data Safe Extension v0.12.0

**Release Date:** 2026-02-17

## Overview

Version 0.12.0 introduces OraDBA extension etc-hook integration for automatic
environment and alias loading.

## Highlights

- **Enabled extension etc hooks** in `.extension`:
  - `load_env: true`
  - `load_aliases: true`
- **Added `etc/env.sh`** for idempotent Data Safe environment initialization.
- **Added `etc/aliases.sh`** for Data Safe helper aliases.

## Details

### Added

- Extension metadata
  - `.extension` now enables OraDBA optional etc hooks using
    `load_env: true` and `load_aliases: true`.
- `etc/env.sh`
  - Initializes `ODB_DATASAFE_BASE`, `DATASAFE_BASE`, and `DATASAFE_SCRIPT_BIN`.
  - Updates `PATH` idempotently when the bin directory exists.
- `etc/aliases.sh`
  - Adds convenience aliases:
    - `dshelp`
    - `dsversion`
    - `dsconcheck`

## Testing

```bash
# Syntax checks for new hook files
bash -n etc/env.sh etc/aliases.sh
```
