# Release Notes v0.19.2

## Highlights

Patch release fixing an `unbound variable` error in `ds_target_delete.sh`
that printed a spurious error line with `set -euo pipefail` on every run.

- **Fixed**: `ds_target_delete.sh` — `SCRIPT_DIR` referenced before definition

---

## Fixed

### `bin/ds_target_delete.sh` — SCRIPT_DIR unbound variable

`SCRIPT_VERSION` was assigned on line 54 using `${SCRIPT_DIR}` before
`SCRIPT_DIR` was defined on line 57. Under `set -euo pipefail` this caused:

```text
ds_target_delete.sh: line 54: SCRIPT_DIR: unbound variable
```

The error was non-fatal (the script completed successfully via the
`|| echo '0.19.1'` fallback), but the error message was printed on every
invocation and is misleading.

**Fix**: Move `SCRIPT_DIR` and `readonly SCRIPT_DIR` above `SCRIPT_NAME`
and `SCRIPT_VERSION`.
